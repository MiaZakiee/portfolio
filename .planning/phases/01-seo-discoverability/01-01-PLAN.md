    {
      "@context": "https://schema.org",

    <link rel="preconnect" href="https://fonts.googleapis.com" />

IMPORTANT notes:
- The Google Fonts `<link>` changes from `rel="stylesheet"` (render-blocking) to `rel="preload" as="style" onload="this.onload=null;this.rel='stylesheet'"` (async). This is the REQ-013 fix.
- The `<noscript>` fallback is REQUIRED for users without JavaScript.
- All URLs are absolute (https://ninocabiltes.dev/...) — never relative for canonical, OG, or sitemap references.
- The `<html lang="en">` is already present — keep it.
- Favicon link tags reference files that Plan 02 will create in `public/`. The link tags must exist regardless.
